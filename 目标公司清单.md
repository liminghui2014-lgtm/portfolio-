## Tier 1：AI Agent / 工作流自动化（最佳匹配）

### Coze（扣子）· 字节跳动
- **投递角色**：产品经理（Agent 方向）/ 社区布道师 / 用户增长
- **为什么你适合**：你是 Coze 上真正把产品用到极致的人——L1/L2 分层调度、夹逼管道架构、APE 审计、125个 Session 的 Swarm 管理。比他们的 PM 更懂用户痛点。你踩过的坑（sessions_send 假成功、sessions_abort 不删除、上下文溢出）就是产品改进清单
- **作品集话术**："我在 Coze 上搭了一套三层架构的 VC 情报系统，日产 S 级研报。这是 L2 工作流的 GitHub 开源版本。你们产品在 Session 管理、工作流调试、错误处理上的短板，我全踩过"
- **入口**：BOSS 直聘搜"扣子"/ Coze 官网 Careers

### Dify
- **投递角色**：Developer Advocate / 社区经理 / 产品经理
- **为什么你适合**：开源 LLM 应用平台的社区运营，需要写文档、讲技术、维护开发者关系。你有 deep-vc-research（MIT 开源），知道怎么把一个工具做成别人能用的东西。你的 SOP Agent Swarm 方法论对他们平台的用户有直接参考价值
- **作品集话术**："我的 deep-vc-research Skill 是开源的 Claude Code 插件。我理解怎么做开源社区的文档、示例和 onboarding。你们平台上关于 Agent 工作流的最佳实践，我可以写"
- **入口**：Dify 官网 Careers / GitHub / 即刻 AI 圈

### 月之暗面（Moonshot AI）
- **投递角色**：产品经理 / 增长 / 解决方案
- **为什么你适合**：你对 Kimi 的产品理解是最深的——写过完整的竞争力拆解。LLM 产品体验、用户画像分析、竞品定位，你都做过了。你也能聊技术（RAG、长上下文、MoE）
- **入口**：官网 / BOSS 直聘 / 即刻 AI 内推群

## Tier 2：AI 应用层 / Agent 基础设施

### 澜码科技（Langx）
- **投递角色**：产品经理 / 解决方案架构师
- **为什么你适合**：专注 AI Agent 基础设施。你的 SOP Swarm（三阶段 Agent 协同）就是他们产品方向的真实用例。你不是"听说过 Agent"，是在生产环境里用过
- **入口**：BOSS 直聘 / 36氪创投平台

### 面壁智能（ModelBest）
- **投递角色**：产品经理 / 商业化 / GTM
- **为什么你适合**：你给他们写过深度研报，对他们的 MiniCPM 端侧策略有独立判断。端侧 AI 赛道需要"懂技术+懂商业"的人——技术型 PM 看不懂商业，销售看不懂端侧推理，你两个都能聊
- **入口**：官网 / BOSS 直聘

### 百川智能
- **投递角色**：产品经理（Agent 方向）
- **为什么你适合**：百川在大模型基础上也在推 Agent 能力。你对 Agent 架构的工程理解（多 Agent 协同、工作流编排、审计机制）正是他们在找的 PM 画像
- **入口**：官网 / 内推

## Tier 3：VC 投后 / 孵化器

### 红杉中国 Value+ 团队
- **投递角色**：投后运营 / 技术顾问
- **为什么你适合**：红杉投后需要帮被投公司解决实际问题——AI 工具落地、数据系统搭建、运营优化。你就是干这个的——帮污水厂做 AI 节能、帮医院做医废就地化、自己搭 Agent 情报系统
- **和面试（投资岗）的区别**：这不是投资分析师岗，是投后赋能岗。更看重实操能力而非建模。面试流程可能不同
- **入口**：红杉官网（查看投后团队招聘）/ LinkedIn

### 奇绩创坛
- **投递角色**：EIR（驻场创业者）/ 技术顾问
- **为什么你适合**：你做的事情（AI Agent 搭建+产品拆解方法论+实体商业经验）是奇绩投的早期 AI 公司最需要的实务指导。你还能帮陆奇团队筛选 AI 赛道的申请者
- **入口**：奇绩官网 / 直接联系团队

### 高瓴 DVC（Deep Value Creation）
- **投递角色**：投后数字化顾问
- **为什么你适合**：高瓴的 DVC 团队专门帮被投企业做数字化转型。你三条腿都站得住：数据分析（硕士+量化）、AI 落地（Agent 系统）、实体产业（节能/医废）
- **入口**：高瓴官网 / LinkedIn
