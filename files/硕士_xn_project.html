XN Project — Facebook 广告数据 EDA 与预测建模

ALY 6080 · 个人项目

ALY 6080
用 Python 和 Tableau 对 ViacomCBS 提供的 Facebook 广告数据进行探索性分析，寻找影响广告互动率的关键变量，构建预测模型。

中文摘要

使用工具：Python (pandas, matplotlib, seaborn, scikit-learn) / Tableau / R

分析方法：EDA（探索性数据分析）、相关性矩阵、热力图、决策树分类模型

分析对象：ViacomCBS 提供的 Facebook 广告数据集（含广告层级数据：impressions, clicks, CPC, CPM, CTR, video views, post reactions 等 20+ 字段）

核心问题：哪些广告特征（受众类型、素材格式、投放时间）最显著影响互动率（CTR/Engagement）？如何用预测模型优化未来广告投放的内容策略？

结论要点：在个人贡献部分，完成了两个数据集的独立 EDA、缺失值处理、相关性可视化，并使用 Tableau 产出交互式仪表盘。发现 video views 与 post reactions 高度相关（r > 0.7），custom audience 类型的广告 CTR 显著高于 lookalike audience。

Introduction

EDA and data cleaning

In this project Python was used to find out the basic statics, missing/null values, and correlation of variables for two data sets. The first step is read the csv files into Python and get basic information about data sets. Then the list of columns of two data sets and one summary table for data types and memory load were displayed. Two data sets contain data in int, object and float form, there are significant amount of null value for customer file. Using describe() function to find the basic statistics for both data sets, also finding the shapes for both data sets.

For customer file, there are significant amount of null value. all the missing/null values were marked. The missing values are distributed mainly from columns ‘description’, ‘clicks’, ‘cpc’ and ‘cpm’. For the look like file, there are only 9 missing/null value from ‘origin_name’. Using dropna() function to drop all the null values for two datasets, then use isnull() function to make sure there are no null values in two data sets.

For this project data cleaning and basic EDA were conducted for two data sets the sponsor has provided, including basic statistics, find the null values, get ride of the null value, define variable’s type and find the correlations between variables. There are still many steps to go, including separate data in to training and testing groups, using tool such as Wordcloud to produce visualization for text variable and apply prediction models such as random forest to solve business problem using the data. Next step is to figure out which prediction model is the best fit to solving problem related to increase ad clicks rate in streaming service, there are many examples I could mirror on sites such as Kaggle. I think I in the future I could use python to produce more visualization results to better communicate my results, also Tableau will be a good choice to display more complicated advanced visualization.

Click rate and Engagement rate

Below is the visualizations of account id and clicks of the data sets. Using this information, it is easy to find which account contributed the highest clicks. The next step is to find out the reasons behind why those account could contribute high clicks.

In order to find what cause the high clicks, another visualization between account id with cpm, ctr and cpc has conducted. It can be concluded from the graph that account that contribute high clicks also contribute high cpc and cpm. There are differences between Cost Per Click(cpc) bidding and Cost Per Thousand Impressions (CPM) bidding, but the mechanism is the similar. This result implies with more investment on bidding strategies that provided by searching machines such as Google Ads could increase the overall click rate for sponsor.

Using similar approach to find out how to engagement rate. First, it is necessary to identify which accounts has contributed the most comments. Account 831180993630429 has contributed the most comments, the next step is to find out the reason. The same account has contributed the most part for all the variables (clicks, comments, cpc and cpm). Therefore, increase marketing investment and increase potential interaction and connection between ads and customers could potentially increase click and engagement rate.

Random Forest Regression

After applying random forest method to the custom audience dataset, it can be concluded that with 20 decision trees, the root mean squared error is extremely high. The result indicates that there were mistakes during data splitting process. If the number of the trees were increased, there will be a decrease in the value of the RMSE with respect to number of estimators. There will be decreases in error value with the increase in number of estimators; therefore, it is important to find a optimal number for the trees and parameters to obtain the best result.
