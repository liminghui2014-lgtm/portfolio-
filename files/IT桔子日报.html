IT桔子 VIP 日报

一级市场每日全景 · 绕反爬的 VIP 数据自动抓取 + Jarvis 风格可视化面板

问题

IT桔子是中国一级市场核心数据库，但 VIP 数据（完整融资记录、机构出手明细、公司深度信息）无法通过公开 API 获取。人工每天登录 IT桔子、逐页翻融资快讯、复制粘贴到 Excel，耗时 40–60 分钟，且容易漏掉关键事件。

技术方案

Chrome CDP 保活机制

核心难点：IT桔子的反爬机制会在无头浏览器或频繁请求时触发验证码。解决方案：不创建新浏览器实例，而是连接到用户已登录的正常 Chrome 实例（CDP 端口 9222），IT桔子看到的还是"正常用户在浏览"。

用户在 Chrome 中手动登录 IT桔子 VIP
        ↓
Chrome 以 --remote-debugging-port=9222 启动
        ↓
itjuzi_keepalive.py 维持登录态
  · 定时刷新页面防止 Session 过期
  · 检测验证码弹窗 → 暂停采集 → 通知用户手动验证
        ↓
itjuzi_daily.py 通过 CDP 连接自动化抓取
  · Phase 1: 市场全景（融资/IPO/并购/独角兽/活跃机构）
  · Phase 2: VC 追踪（红杉中国/深创投/经纬/高瓴最新出手）
  · Phase 3: 公司深度（估值 $1B+ 公司团队+融资历史）
        ↓
generate_pdf.py 输出结构化 PDF 日报

三阶段产出

阶段内容信源

Phase 2 · VC 追踪红杉中国、深创投、经纬创投、高瓴创投 最新出手 + 投资逻辑分析机构主页 + 被投公司交叉验证
Phase 3 · 公司深度估值 $1B+ 公司：团队背景、融资历史、竞品对比公司详情页 + 工商信息

Jarvis 可视化面板

Iron Man 风格的 HUD 大屏：黑屏待机 → 语音唤醒动画 → 实时数据面板（融资总额 / 活跃赛道 / 机构出手热力图 / 地域分布）。使用 HTML + CSS animation + edge-tts 语音合成，用于抖音内容素材。
