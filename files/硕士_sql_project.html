edX 在线教育平台 — 数据库设计与SQL查询

ALY 6030 · 个人项目

ALY 6030
为 edX 在线课程平台设计规范化数据库模型（ER图 → 建表 → 查询优化），编写复杂 SQL 查询回答业务问题：用户行为分析、课程完成率、人口统计分布。

中文摘要

使用工具：MySQL Workbench / SQL

分析方法：数据库规范化设计（1NF→2NF→3NF）、ER 图、复杂 SQL 查询（JOIN、子查询、聚合函数、HAVING）

分析对象：edX 在线教育平台的用户选课数据集（含用户信息、课程信息、成绩记录、证书状态等多表关联数据）

核心问题：如何将原始的"扁平表"规范化到第三范式（3NF）？如何用 SQL 回答业务问题——哪些课程完成率最高？不同学历背景的用户选课偏好有何差异？证书获取率与课程类型是否相关？

结论要点：成功将原始 14 列扁平表拆分为 3 张规范化表（users, courses, enrollments），消除了数据冗余。通过 6 组 SQL 查询回答了课程完成率排名、用户行为分析、人口统计分布等业务问题。掌握了 JOIN 多表关联、子查询嵌套、GROUP BY + HAVING 的实战应用。
